## Getting Started

### Prerequisites

- Odin compiler (version: "dev-2026-02-nightly:b942f72")
- OpenGL 4.6

### Installation

Clone the repo
Run with [Odin](https://odin-lang.org/)

```powershell
git clone https://github.com/hugo3125soko312/odinGL.git
cd odinGL
odin run .
```
